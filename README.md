
# Artha - Budgeting App

A mobile application that helps users track income, expenses, manage budgets, and monitor financial habits.
The app focuses on simplicity, clean UI, and fast daily financial input.


## Screenshots

![App Screenshot](https://dummyimage.com/468x300?text=App+Screenshot+Here)


## Target

* Individual users
* Students / workers
* People who want simple budgeting
* Not for accounting professionals
## Tech Stack
Frontend:

* Flutter
* State Management: Riverpod

Local Database:

* SQLite via Drift ORM

Backend:

* Supabase
* PostgreSQL Database
* Supabase Auth (future multi user)
* Supabase Storage (receipt images)

Charts:

* fl_chart package
